# Custom StringBuilder with Undo Feature

## 📌 Overview
This project is a custom implementation of a simplified StringBuilder in Java with an additional `undo()` feature using the Snapshot (Memento) design pattern.

## 🚀 Features
- Append strings
- Delete substring
- Undo last operation
- Maintains history using stack

## 🧠 Concept Used
- Snapshot (Memento) Pattern
- Stack (LIFO)
- Mutable strings

## 🛠️ Tech Stack
- Java

## 📂 Example

Input:
Hello → append " World" → undo

Output:
Hello World
Hello

## 🎯 Learning Outcome
- Understanding internal working of StringBuilder
- Designing undo functionality
- Applying design patterns in real use-case
